## Лабораторная работа №2. Настроить OSPF для Underlay сети.


1. Настроите OSPF в Underlay сети, для IP связанности между всеми сетевыми устройствами.
2. Зафиксируете в документации - план работы, адресное пространство, схему сети, конфигурацию устройств
3. Убедитесь в наличии IP связанности между устройствами в OSFP домене

План работы.
1. Проектируем сеть ospf, исходя из условия задачи у нас 1 POD, поэтому одна area.
   Основные пораметры сети OSPF:
    - № процесса 100;
    - area 0.0.0.0
    - bfd default;
    - passive-interface default;
    - router-id;
   __________________________________________________________________________________________
   | name     | Leaf-1      | Leaf-2      | Leaf-3      | Spine-1      | Spine-2             |
   |----------|-------------|-------------|-------------|--------------|---------------------|
   |router-id | 10.16.1.250 | 10.16.2.250 | 10.16.3.250 | 10.16.17.250 | 10.16.18.250        |
   ___________________________________________________________________________________________
  - bfd на интеррфейсе в сторону соседа 100 ms x 3;
  - ip ospf authentication-key: POD1;
2. Схема сети [Схема_OSPF.pdf](./Схема_OSPF.pdf).
